## 파일 업로드 규칙

### 파일 명

- 문제 풀이 파일은 **아래 형식을 반드시 지켜서 업로드**해 주세요.
- !! 형식에 어긋난 pr은 반려됨
  - 추후 봇 도입 예정

- 식별자는 알아서 정하기
  - 영어 소문자 및 숫자만 가능 (특수문자 사용 불가)
  - 숫자로 시작하지 않기
  - 한 단어 또는 약어로  

- 예
  ```
  백준문제번호_식별자.py
  ```

### 업로드 해야 할 폴더
- `주차/오늘` 날짜 폴에 올리기
- 예를 들어 1주차면 [week1](/week1/)안에 [week1/260310](/week1/260310)과  같이 해당 일에 맞는 문제 폴더에 올리기
- 해당 폴더에 가보면 문제가 [마크다운](/week1/260310/README.md)안에 기제되어 있음
